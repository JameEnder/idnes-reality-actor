## Idnes Reality Actor

Scrape Idnes Reality using Crawlee (probably on the Apify platform).

You can scrape properties for a certain location and get all the details about them.

## Example input

```json
{
	"location": "Praha",
	"maxResults": 50,
	"dealType": "rent",
	"propertyType": "house",
	"maxPrice": 40000
}
```
